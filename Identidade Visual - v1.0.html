07 Regras de Uso
✓
Use o preto como base dominante
O preto é a cor principal. O dourado é o acento — nunca o contrário.
✗
Não inverta as proporções
Evite fundos totalmente dourados com texto preto como padrão de layout.
✓
Fonte: Inter sempre
Toda a comunicação usa Inter. Headlines em 900, corpo em 400, nunca misture outras famílias.
✗
Não use o logo em fundos coloridos
O logo funciona apenas em fundo preto (#111) ou creme (#F5F2EE).
✓
Minúsculas sempre no nome
"café com campinas" — sempre em letras minúsculas, sem exceção.
✗
Não distorça o ícone
O ícone da xícara deve ser sempre proporcional. Nunca esticado ou comprimido.
